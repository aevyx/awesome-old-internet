## 1. Platform Layout & UI Restoration

Projects designed to revert modern websites back to their beloved, classic interfaces.

### YouTube

- **[YouTube Hitchhiker Restoration](https://vorapis.pages.dev)**: Userscript restoring the pre-2017 Hitchhiker layout for YouTube.
- **[StarTube](https://github.com/lightbeam24/StarTube)**: Browser add-on for selecting YouTube layouts from 2008 to present.

### Google Search & Services

- **[GPlex (Old Google Frontend)](https://greasyfork.org/en/scripts/492193-gplex-old-google-frontend)**: Userscript restoring early 2010s Google search layouts.
- **[Google Gbar Restoration](https://github.com/CallyHam/Google-Gbar/)**: Userscript re-implementing the classic Google shortcut bar.

### Social Media & Messaging

- **[OldTwitter](https://github.com/dimdenGD/OldTwitter)**: Browser add-on reverting Twitter (X) to its pre-2023 design.
- **[OldTweetDeck](https://github.com/dimdenGD/OldTweetDeck)**: Add-on/app restoring the original TweetDeck interface.
- **[OldTwitter 2013 Fork Addon](https://userstyles.world/style/22460/some-oldtwitter-2013-fork-read-description)**: Userstyle for the 2013 Twitter aesthetic.
- **["X" to Twitter Reverter](https://userstyles.world/style/15959/)**: Userstyle reverting X branding to classic Twitter.
- **[DTM-XX Theme](https://github.com/XYZenix/XYZenixThemes)**: Discord theme restoring pre-2021 looks.
- **[Escargot MSN Messenger](https://escargot.chat/)**: Server emulator for MSN Messenger.
- **[OpenSteamClient](https://github.com/OpenSteamClient)**: Client mod restoring older Steam client versions.
- **[AIM Phoenix](https://iwarg.ddns.net/aim/)**: Revival of AOL Instant Messenger (AIM) servers and clients.

### Reddit

- **[Classic Reddit Rewrite](https://userstyles.world/style/21663/classic-reddit-rewrite)**: Userstyle restoring pre-2017 Reddit iconography.
- **[Reddit Classic](https://userstyles.world/style/7997/reddit-classic)**: Userstyle replicating the 2006-era Reddit look.



## 2. Browser & OS Skinning/Theming

Projects focused on reviving the look and feel of classic operating systems and browser versions.

### Chromium Forks & Browsers

- **[Supermium](https://github.com/win32ss/supermium)**: Chrome fork for older Windows, with Aero and XP themes.
- **[Superbird](https://superbird-browser.com/)**: Chromium fork with classic UI/UX elements.

### Firefox Themes

- **[Geckium](https://github.com/angelbruni/Geckium)**: UserChrome theme mimicking old Google Chrome UI in Firefox.
- **[Echelon](https://echelon-theme.github.io/)**: UserChrome theme restoring old Firefox/Mozilla suite aesthetics.
- **[Photon/Australis UI Restorer](https://github.com/Aris-t2/CustomCSSforFx)**: UserChrome styles to remove Proton UI and restore Photon/Australis.
- **[Netscape 4.7 Theme](https://addons.thunderbird.net/en-US/firefox/addon/netscape-4-modern/)**: UserChrome theme replicating Netscape Navigator.



## 3. The Ultimate Old Web Experience

Essential tools, archives, and resources for truly immersing yourself in the awesome old internet.

### Web Archives & Emulators

- **[The Wayback Machine](https://archive.org/web/)**: Archive of the internet since 1996.
- **[GeoCities Archive](https://geocities.ws/)**: Archive of millions of GeoCities pages.
- **[Cameron's World](http://www.cameronsworld.net/)**: Art project using GeoCities elements.
- **[The CERN Project](http://info.cern.ch/)**: The first website, preserved in its original form.
- **[Oldweb.today](http://oldweb.today/)**: Emulator for browsing archived content in old browsers.
- **[Lynx Browser](https://lynx.invisible-island.net/)**: Terminal-based, text-only web browser.

### Design & Development Resources

- **[Web-Safe Color Palette](https://www.w3schools.com/colors/colors_picker.asp)**: 216 colors for old 8-bit monitors.
- **[Animated GIF Archives](https://gifcities.org/)**: Massive collection of vintage GIFs.
- **[MIDI File Archives](https://www.classicalarchives.com/midi.html)**: MIDI files for background music.
- **[W3C HTML 3.2 Spec](https://www.w3.org/TR/REC-html32)**: HTML standards for Web 1.0.
- **[98.css](https://jdan.github.io/98.css/)**: CSS library for Windows 98/Web 1.0 aesthetics.
- **[Win95.css](https://kristofferandreasen.github.io/win95.css/)**: CSS toolkit for Windows 95 look.

### Gaming & Flash Preservation

- **[Flashpoint Archive](https://bluemaxima.org/flashpoint/)**: Game preservation project for Flash, Shockwave, and Java games.
- **[Ruffle](https://ruffle.rs/)**: Open-source Flash emulator.
- **[Newgrounds Classic Skin](https://userstyles.world/style/11013/newgrounds-classic)**: Userstyle for classic Newgrounds look.
- **[Neocities](https://neocities.org/)**: Modern platform for building and hosting classic-style personal homepages.
- **[Nekoweb](https://nekoweb.org/)**: Another one like Neocities

### Webrings & Blinkies

- **[Yesterweb Webring](https://yesterweb.org/webring/)**: Community of sites dedicated to the old web spirit.
- **[Sadgrl Webring](https://sadgrl.online/webring/)**: Another active webring for personal, old-style sites.
- **[Webring Directory](https://webring.directory/)**: Directory of active webrings.
- **[Blinkies.Cafe](https://blinkies.cafe/)**: Huge archive of classic blinkies and web graphics.
- **[GifCities](https://gifcities.org/)**: Searchable archive of GeoCities GIFs.
- **[88x31 Button Archive](https://cyber.dabamos.de/88x31/)**: Collection of classic 88x31 web buttons.
